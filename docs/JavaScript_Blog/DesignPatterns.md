[toc]









##  不同的设计模式

**1.创建型模式**

这一类设计模式的目的是用于创建对象。

**2.结构型模式**

这一类设计模式的目的是优化不同类、对象、接口之间的结构关系。

**3.行为型模式**

这一类设计模式的目的是更好地实现类与类之间的交互以及算法的执行。
